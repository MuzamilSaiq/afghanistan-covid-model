# afghanistan-covid-model
Daily country-level COVID-19 case and fatality data from Kaggle (2020–2024); used for descriptive analysis only.

## Descriptive Analysis of COVID-19 Case Reporting

Descriptive analysis of daily reported COVID-19 cases and deaths.
No forecasting or causal inference is attempted.

**Data:** Daily COVID-19 Data (2020–2024), Kaggle (abdoomoh).  
**Methods:** EDA, visualization, smoothing, GLM diagnostics

**Reference:**  
Zhou et al. (2020). *Clinical course and risk factors for mortality of adult inpatients
with COVID-19 in Wuhan, China*. **The Lancet, 395**(10229), 1054–1062.
